# JS Practice

前端工程師轉職中，小組專案開始前兩週的練習。

這兩週的練習目標：
- 把課堂學過的 JavaScript 核心語法練到能獨立運用
- 建立從資料處理到畫面呈現的完整開發能力
- 熟悉 Git / GitHub 協作工作流程，為小組專案做準備

## 技術範圍

**JavaScript**
- 陣列方法：filter、map、reduce、find、findIndex
- 非同步：fetch、axios、async/await、try/catch
- DOM 操作：querySelector、addEventListener、insertAdjacentHTML
- 資料儲存：localStorage、JSON.parse / JSON.stringify
- 物件導向：工廠函式、class、this、prototype

**CSS**
- Flexbox、Grid
- RWD：media query 響應式設計
- Tailwind CSS、Bootstrap

**工具**
- Git：branch、merge、pull request
- GitHub：版本控制與協作流程
